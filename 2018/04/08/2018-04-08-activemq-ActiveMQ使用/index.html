ActiveMQ使用

2018-04-08 后端 ActiveMQ 0 Comments

ActiveMQ 是一个apache提供的一个消息队列服务,为我们在生产中提供了一个很好的服务器,我们不需要使用Java BloingQueue,使用这个服务就能很好的应对各种生产场景。

下载

我们需要到apache官网上下载ActiveMQ,官网地址http://activemq.apache.org/activemq-5145-release.html,我这边是ubuntu操作系统,下载尾缀是tar.gz,如果是windows下载zip

配置activemq.xml

解压下好的软件

进入apache-activemq-5.14.5/conf找到activemq.xml,将其中的0.0.0.0修改成127.0.0.1

启动

进入到apache-activemq-5.14.5/bin 目录下执行命令

$ ./activemq start

打印下列语句就是开启成功了

访问localhost:8161/admin。
